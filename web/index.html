Anonymous Git Contributions

Contribute to open source projects anonymously. gitGost strips identifying information from your commits and creates pull requests on your behalf.

View on GitHub

Usage

Add gitGost as a remote and push anonymously:

Last login:
➜ ~ git remote add gitgost https://gitgost.leapcell.app/v1/gh/owner/repo
➜ ~ git checkout -b my-feature
➜ ~ echo "my changes" >> file.txt
➜ ~ git commit -am "Add feature"
➜ ~ git push gitgost my-feature:main
# gitGost will anonymize your commits and create a PR
➜ ~

Features

Privacy First

Strips author names, emails, and timestamps from commits

Automated PRs

Automatically creates pull requests on GitHub

Secure

Rate limiting, size limits, and input validation

Simple

Just add a remote and push - no complex setup

Why gitGost?

Just because you support open source doesn't mean you want your name forever associated with every project you touch.

Privacy

GitHub's social coding means every commit is visible forever. Sometimes you want to contribute without the personal association.

Controversial Projects

Government tracking, corporate restrictions, or simply wanting to prove a point - contribute privately when needed.

Simple & Trustworthy

Adding a git remote is easier than managing multiple GitHub accounts. Built with Go, minimal dependencies, and open source.

About

gitGost is built for developers who value privacy. It uses go-git for Git operations and creates anonymous contributions to GitHub repositories.

Built with Go 1.22, AGPL-3.0 licensed, and designed for minimalism and security.
